설치 방법

  1. 출처를 알 수 없는 앱 설치 허용
    설정 → 보안 → 출처를 알 수 없는 앱 설치 허용
  2. APK 파일 다운로드
    위의 'APK 다운로드' 버튼을 눌러 파일을 다운로드합니다.
  3. 다운로드한 APK 파일 실행
    다운로드 폴더에서 APK 파일을 찾아 설치를 진행합니다.
  4. 설치 완료
    앱 실행 후 테스트를 진행해주세요.
